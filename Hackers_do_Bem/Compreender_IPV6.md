# Tipo de Endereço
Métodos de comunicação: **UNICAST; ANYCAST; BROADCAST E MULTICAST;**


## UNICAST
É o tipo de comunicação feita de um para um. é um endereçamento para um pacote feito a um único destino.
**1 para 1**

## ANYCAST
Feita de uma interface para muitas outras, contanto que elas estejam na mesma região. Cada dispositivo anuncia o mesmo endereçço de rede.
**1 Para um grupo de interfaces próximas**

## MULTICAST
É quando um dispositivo se comunica com alguns receptores. É o método de transmissão de pacote de dados para múltiplos utilizado pelo IPv6.
**1 Para um grupo de interfaces**

## BROADCAST
É o método de transferência simultânea para diversos receptores. É utilizado especificamente no IPv4.

**1 Para TODOS**
